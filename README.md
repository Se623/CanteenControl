# CanteenControl
Автоматизированная информационная система школьного питания.\
Имеет 3 вида профиля: ученик, профиль, администратор.\
Позволяет организовать распеределение и закупку блюд, сформировать отчёт столовой.\
## Установка
- Скачиваете репозиторий
- Запускайте server.py с помощью Python
- Подключайтесь по адресу http://127.0.0.1:5000 с помощью веб браузера
### Важно!!! Регистрация для поваров и администраторов не предусмотрена. Для создания хэша пароля пользуйтесь скриптом hashgen.py (Запустите скрипт, введите пароль, и скопируйте хэш пароля в базу данных)
